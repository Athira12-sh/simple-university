# how to use git and github

change filter -> commit to localreport with message -> Puch to Github
1.after changing files you have to commit the changes to local repository
2.to commit changes from VSCODE IDE use the left icon ctrl+shift+G
3.for each commit you need message
4. write the message in the text box and click commit.Message should be meaningful
5.after commiting the changes are stored in local repository not github
6.To apply changes to github repository you have to push the changes to github by clicking the push icon(upward arrow)
7.you can check the changes on github
